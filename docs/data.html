Data využívaná v kurzu

V kurzu jsou primárně využívaná data o Evropských zemích, získáná z Eurostatu.

Dataset je možné stáhnout v několika formátech:

  • Comma separated values formát je ke stažení zde (klikněte pravým tlačítkem a zvolte Uložit odkaz jako…/Save link as…).

  • RDS formát zde.

Data obsahují následující proměnné:
